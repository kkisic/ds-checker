# ds-checker
Death Strandingの指名なし依頼のクリア状況を記録するツールです.

LoL埋め等に活用してください.

ツール上部の入力欄から複数依頼を同時に更新可能です. (カンマ区切りさえ守られていればスペース等はよしなに処理されます)
